# Testowanie Aplikacji Mobilnych - Portfolio QA 🚀

**Autor:** Kacper Przepiórka (97104)

## O projekcie
Repozytorium zawiera kompletny proces automatyzacji testów mobilnych zrealizowany w ramach zajęć laboratoryjnych. Projekt udowadnia umiejętność budowy cyklu testowego (CI/CD) - od zarządzania infrastrukturą po przejrzyste raportowanie.

## Zrealizowane Bloki (1-10)
* **Bloki 1-2:** Konfiguracja ADB, zarządzanie emulatorami i kontrola wersji Git.
* **Blok 3:** Uruchomienie serwera Appium w kontenerze Docker (Docker-Compose).
* **Blok 4-5:** Inspektor Appium, selektory elementów i pierwsze skrypty automatyczne E2E.
* **Blok 6:** Wdrożenie wzorca projektowego Page Object Model (POM).
* **Blok 7:** Testowanie zaawansowanych gestów (Swipe) i testy przerwań.
* **Blok 8:** Statyczna analiza bezpieczeństwa (SAST) i audyt pliku Manifest.
* **Blok 9:** Testowanie API backendowego metodą headless i testy mostka hybrydowego.
* **Blok 10:** Wdrożenie Allure Reports i automatyczny Pipeline sklejający cały projekt.

## Technologie
* **Język:** Python 3
* **Narzędzia:** Appium, Docker, Pytest, Allure, Git, ADB, uiautomator2
